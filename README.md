# Automated Supply Chain Exception Engine (SAP + n8n)

### 🚀 Overview
This project automates the identification of shipment delays by joining SAP Purchase Order data with live logistics tracking, eliminating the need for manual T-code exports and VLOOKUPs.

### 🛠️ The Tech Stack
* **n8n:** Workflow automation & ETL logic.
* **SAP (Simulated):** PO data source.
* **Slack API:** Real-time exception alerting.
* **Excel/Google Sheets:** Final BI-ready data destination.
